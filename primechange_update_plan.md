# PRIME CHANGE ホームページ最新版アップデート計画

作成日: 2026-05-22  
対象サイト: https://primechange.co.jp/  
前提: 現行サイトの公開内容を理解したうえで、内容、UI、SEO、技術基盤、運用体制を最新版に更新する計画。

## 1. アップデートの目的

PRIME CHANGEのホームページを、単なる会社紹介サイトから「ホテル清掃・運営支援の問い合わせ獲得」「FC加盟候補者の獲得」「採用導線の強化」「企業信頼性の向上」を同時に担うサイトへ更新する。

特に、ホテル清掃会社を探しているホテル担当者に対して、以下を短時間で伝えられる状態を目指す。

- 何を提供しているか
- 他社と何が違うか
- どのホテルで実績があるか
- 導入すると何が改善するか
- どのように相談・問い合わせすればよいか

## 2. 優先度別ロードマップ

### Phase 1: 緊急点検・基盤アップデート

目安: 1-2週間

- WordPress本体、テーマ、プラグイン、PHP、MySQL/MariaDBの現状バージョンを棚卸しする。
- 現行サイトの完全バックアップを取得する。
- ステージング環境を作成し、本番と同じ構成を再現する。
- WordPress 5.8.13から最新版への更新可否を検証する。
- 2026年5月22日時点でWordPress公式ダウンロードページはWordPress 7.0を案内しているため、WordPress 7.0を目標バージョンとして検証する。
- WordPress公式の推奨環境であるPHP 8.3以上、MySQL 8.0またはMariaDB 10.6以上を満たせるか確認する。
- 問い合わせフォーム、お悩み相談室、FC資料請求フォームの動作確認を行う。
- サンプルページ、テスト投稿、Hello worldなど不要公開物があれば非公開または削除する。
- `/service/staffing/` の404/noindex、`/philosophy/` と `/company/philosophy/` のURL揺れを整理する。

成果物:

- 技術棚卸し表
- ステージング環境
- バックアップ手順
- 更新リスク一覧
- URL修正方針

## 3. Phase 2: 情報設計と導線整理

目安: 2-3週間

### 3.1 グローバルナビの整理

推奨ナビゲーション:

- ホテル清掃サービス
- ホテル運営支援
- 導入実績・お客様の声
- 会社案内
- メンバー
- FC募集
- 採用情報
- お問い合わせ

「人材サービス」は、実際に提供するサービスとして掲載するならページを新規整備し、提供しないならメニューから削除する。

### 3.2 問い合わせ導線の再設計

問い合わせ導線は目的別に分ける。

- ホテル清掃を相談したい
- ホテル運営支援を相談したい
- FC加盟について知りたい
- 採用について知りたい
- 清掃業者選びの悩みを相談したい

各ページ下部に、ページ内容に合ったCTAを配置する。

### 3.3 サイト構成案

- トップページ
- ホテル清掃サービス
- ホテル運営支援
- 導入実績
- お客様の声

- 会社案内
- 私たちの信念
- メンバー紹介
- FC募集
- 採用サイトへの導線
- ブログ
- お問い合わせ

## 4. Phase 3: コンテンツ刷新

目安: 3-5週間

### 4.1 トップページ

トップページの第一画面で、以下を明確にする。

- ホテル清掃と運営支援の専門会社であること
- 契約客室数、対応エリア、導入実績などの主要数字
- PRIME CHANGEの強みである人材集客、教育、チームワーク
- ホテル担当者向け問い合わせCTA
- FC候補者向けCTA

推奨セクション:

- ファーストビュー
- 選ばれる理由
- ホテル清掃サービス概要
- ホテル運営支援概要
- 導入実績・契約客室数
- お客様の声
- よくある相談
- メンバー・信念
- お問い合わせ

### 4.2 ホテル清掃サービスページ

現行の強みを活かしつつ、以下を追加する。

- 対応範囲: 客室清掃、共用部清掃、リネン、備品補充、点検、報告など
- 導入までの流れ
- 品質管理体制
- 教育制度の具体例
- 現場責任者の役割
- トラブル対応、改善提案の仕組み
- 実績ホテルの更新
- よくある質問

### 4.3 ホテル運営支援ページ

現行のSNS PR、販促、差別化支援に加え、ホテル担当者が成果をイメージできるようにする。

- 支援メニュー一覧
- SNS運用支援の範囲
- インフルエンサー施策の実施フロー
- 販促品・ギフト券・回数券の活用例
- 直接予約増加に向けた施策例
- 清掃品質と口コミ改善の関係
- 導入事例

### 4.4 お客様の声・導入事例

現在の2件を軸に、定期的に追加する。

掲載フォーマット:

- ホテル名
- 導入前の課題
- PRIME CHANGEを選んだ理由
- 導入後の変化
- 現場で評価している点
- 今後期待すること
- 写真または担当者コメント

### 4.5 FC募集ページ

現行ページは訴求力が高い一方、収益表現や加盟条件は慎重な整備が必要です。

更新項目:

- 年商表現の根拠、注記、条件を明確化
- 加盟費用、研修内容、サポート範囲を表形式に整理
- 加盟後の収益モデル例を複数パターンで提示
- 向いている人、向いていない人を明記
- 加盟までの流れをより具体化
- 説明会予約または資料請求フォームを改善
- よくある質問を増やす

### 4.6 採用導線

採用サイトへの外部リンクだけでなく、公式サイト側にも以下を短く掲載する。

- 働く人の多様性
- キャリアパス
- 教育体制
- メンバー紹介との連動
- 採用サイトへの明確なCTA

## 5. Phase 4: UI/UX刷新

目安: 3-6週間

### 5.1 モバイル対応

現行HTMLではviewportが `width=1280` になっているため、スマートフォンでの自然な閲覧体験を再設計する。

対応方針:

- `width=device-width, initial-scale=1` を前提にレスポンシブ化
- 主要CTAをスマートフォンで押しやすくする
- フォーム入力項目を簡潔にする
- メニューを目的別に整理する
- 画像、動画、アニメーションの表示負荷を抑える

### 5.2 デザイン方針

ホテル業界向けBtoBサイトとして、清潔感、信頼感、スピード感を重視する。

- 白、黒、赤を基調にしつつ、余白と写真で清潔感を出す
- 現場写真、スタッフ写真、ホテル実績写真を増やす
- 装飾よりも実績、品質、導入フローの見やすさを優先する
- CTAは「問い合わせ」「資料請求」「相談する」の3種を明確に使い分ける

## 6. Phase 5: SEO・構造化データ・計測

目安: 2-4週間

### 6.1 SEO

ページごとに検索意図を整理する。

- ホテル清掃 東京
- ホテル 客室清掃 外注
- ホテル清掃 業者
- ホテル 清掃会社 変更
- ホテル清掃 フランチャイズ
- 客室清掃 人材不足
- ホテル SNS 集客

対応項目:

- titleタグ、meta descriptionの再設計
- H1-H3構造の整理
- 画像altの最適化
- 内部リンク設計
- パンくずの正規化
- サイトマップ確認
- 不要ページのnoindexまたは削除

### 6.2 構造化データ

追加・確認する構造化データ:

- Organization
- LocalBusiness
- BreadcrumbList
- Service
- FAQPage
- ArticleまたはBlogPosting

### 6.3 計測

Google Tag Managerは導入済みのため、以下のイベントを設計する。

- 問い合わせボタンクリック
- フォーム送信完了
- FC資料請求
- 採用サイトクリック
- お悩み相談室送信
- 電話番号クリック
- 動画再生

## 7. Phase 6: セキュリティ・運用体制

目安: 継続運用

### 7.1 セキュリティ

- WordPress本体、テーマ、プラグインの月次更新
- 自動バックアップ
- 管理画面ログイン制限
- 二要素認証
- 不要ユーザー、不要プラグイン、不要テーマの削除
- reCAPTCHAまたは代替スパム対策
- SSL、HSTS、セキュリティヘッダー確認
- PHPエラーログ、フォーム送信ログの監視

### 7.2 運用

月次で確認する項目:

- ブログ更新
- お客様の声追加
- 清掃実績の更新
- メンバー情報の更新
- FC募集エリアの更新
- フォーム送信テスト
- Search Consoleの検索クエリ確認
- AnalyticsのCV確認

## 8. 更新時の注意点

- WordPress 5.8.13からWordPress 7.0への更新は差分が大きいため、本番で直接更新しない。
- 古いテーマやプラグインがWordPress 7.0、PHP 8.3以上に対応していない可能性がある。
- フォーム、確認画面、完了画面、メール送信、スパム対策は重点確認する。
- 現在のURL資産を壊さないよう、変更が必要な場合は301リダイレクトを設定する。
- FC募集ページの売上表現は、景品表示法や契約説明上のリスクを考慮して、条件・根拠・注記を明確にする。
- 顧客名、ホテル名、担当者写真は掲載許諾を確認する。
- 多国籍スタッフの紹介は、本人同意と表記の正確性を確認する。
